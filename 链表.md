# 单链表

用结构体实现单链表，

```c++
struct ListNode{
    int val;
    ListNode * next;
    ListNode(int x):val(x),next(nullptr){};
    ListNode(int x, ListNode * l):val(x), next(l){};
}
```





用数组实现单链表，head表示头结点的下标，e[i]表示节点i的值，ne[i]表示节点i的next的指针，idx存储当前已经使用到了哪个点

```c++
//全局变量
int head, e[N], ne[N], idx;
```





结构：

​		head 代表指针，指向单链表的第一个节点， idx 指向的是当前可插入的位置，相当于 new 的操作。



数组实现的单链表中，数组下标表示指针，idx 表示下一个未被使用的地址也就是数组下标，e[idx] 表示节点存储的值，ne[idx] 表示节点存储指向下一个节点的指针也就是数组下标。



## 初始化

单链表的初始化时将head指针指向一个空集。可以将head节点看做一个虚拟节点，意味着head指针为一个标志位，head指针始终指向链表的头部。

初始化可以在翻转链表时使用

```c++
void init(){
    head = -1;
}
```







## 查找

单链表查找操作时间复杂度通常为O(n)，需要遍历整个链表才能找到需要的值

用结构体实现单链表的查找操作为

```c++
Listnode* find(int x){
    for(Listnode *i = head; i; i = i->next){
        if(i->val == x) return i;
    }
    return nullptr;
}
```



用数组实现单链表的查找操作为

```c++
int find(int x){
    for(int i = head; i != -1; i = ne[i]){
        if(e[i] == x) return i;
    }
    return -1;
}
```









## 插入

单链表插入操作，大部分情况都是将新的值插到链表的头部，将head指针指向新插入的节点。



用结构体实现单链表的插入操作为

```c++
void add_head(int x){
    Listnode *l = new Listnode(x, head->next), head = l;
    
}  
```





用数组实现单链表的插入操作为

```c++
void add_head(int x){
  e[idx] = x, ne[idx] = head, head = idx++;  
    
}
```

## 翻转

新建一个空指针 rehead，指向的是翻转后链表的第一个值，由于一开始翻转的链表为空，故为空指针。之后以 while(head) 为循环条件，当 head 不为空时，将 head 的下一个节点用临时变量 tmp 存储，head 节点指向 rehead，rehead 指向 head， head 指向 tmp 节点。其中 head 节点指向 rehead 节点相当于向翻转后的链表插入值，用临时变量 tmp 存储 head 节点相当于删除原链表中的第一个值。



用结构体实现单链表的翻转操作为

```c++
void reverse(Listnode *head){
    Listnode *newhead = nullptr;
    while(head){
        Listnode *tmp = head->next;
        head->next = newhead;
        newhead = head;
        head = tmp;
    }
    head = newhead;
}
```
